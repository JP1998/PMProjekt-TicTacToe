Zwei Spieler gegeneinander

Die Namen der Spieler können Sie in der oberen rechten Ecke sehen.

Spiel neu starten

Um ein Spiel neu zu starten, können Sie einfach den "Spiel zurücksetzen" Button benutzen. Um neue Namen zu vergeben müssen Sie das Spiel vorher löschen.

Farben und Zeichen

Der Spieler, der zuerst am Zug ist hat immer das rote Kreuz. Der zweite Spieler immer den grünen Kreis.

Gewinnkondition

Um das Spiel zu gewinnen müssen Sie vor Ihrem Gegenspieler drei Felder in einer Reihe belegen.
